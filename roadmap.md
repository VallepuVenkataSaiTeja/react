# 🚀 Phase 1: Web Fundamentals (Before React)

You must be comfortable with:

### ✅ 1. HTML

* Semantic tags
* Forms & validation
* Accessibility basics

### ✅ 2. CSS

* Flexbox
* Grid
* Responsive design
* Animations
* CSS variables
* Tailwind (optional but popular)

### ✅ 3. JavaScript (Very Important)

Master modern ES6+:

* `let`, `const`
* Arrow functions
* Destructuring
* Spread / Rest operators
* Array methods (`map`, `filter`, `reduce`)
* Promises
* `async/await`
* Modules (`import/export`)
* DOM basics
* Event loop basics

If JS isn’t strong, React will feel difficult.
